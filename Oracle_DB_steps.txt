
Oracle AI Database Setup and Connection Guide
=============================================

1. Install Oracle AI Database Free (26ai)
-----------------------------------------
- Download Oracle AI Database Free from the official Oracle website.
- Run the installer as Administrator.
- Choose a typical installation.
- Set an admin password.
- The default service name is usually FREEPDB1.
- Listener runs on port 1521.

2. Verify Installation
----------------------
Open Command Prompt and run:

    lsnrctl status

You should see the service FREEPDB1.

3. Connect as SYSDBA
--------------------
In Command Prompt:

    sqlplus / as sysdba

If successful, SQL> prompt appears.

4. Switch to Pluggable Database
-------------------------------

    ALTER SESSION SET CONTAINER = FREEPDB1;

5. Create Application User for RAG
----------------------------------

    CREATE USER rag_user IDENTIFIED BY "YourPassword" ACCOUNT UNLOCK;
    GRANT CREATE SESSION TO rag_user;
    GRANT CREATE TABLE TO rag_user;
    GRANT UNLIMITED TABLESPACE TO rag_user;

6. Test Connection
------------------

    CONNECT rag_user/YourPassword@localhost:1521/FREEPDB1

If "Connected" appears, user setup is correct.

7. Enable Vector Memory (Required for Vector Search)
----------------------------------------------------

    ALTER SYSTEM SET VECTOR_MEMORY_SIZE = 500M SCOPE=SPFILE;
    SHUTDOWN IMMEDIATE;
    STARTUP;

Verify:

    SHOW PARAMETER vector_memory_size;

8. Python Connection String (.env)
----------------------------------

    ORACLE_USER=rag_user
    ORACLE_PASSWORD=YourPassword
    ORACLE_DSN=localhost:1521/FREEPDB1

9. Connecting from Python
-------------------------

    import oracledb
    conn = oracledb.connect(
        user="rag_user",
        password="YourPassword",
        dsn="localhost:1521/FREEPDB1"
    )

This connection is used by LangChain OracleVS to store embeddings and perform semantic vector search.
